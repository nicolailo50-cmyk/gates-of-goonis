# Gates of Goonis

A "Gates of Olympus"-style tumble slot, reskinned as an inside joke among friends.
Play credits only — **no real money, no real gambling.**

- Single-page game, runs entirely in the browser (no server, no backend).
- 6×5 tumbling reels, football multiplier orbs, 15 free spins with a persistent multiplier.
- Buy Free Spins for 100× the base bet.
- Base RTP ~96.5%, bought-bonus RTP ~98% (tuned via Monte Carlo simulation).

## Run locally
Just open `index.html` in any browser.

## Hosting
Static files only — host `index.html` + the `assets/` folder anywhere (e.g. GitHub Pages).
